# Pirate Ocean Assets

This folder contains all game assets organized by type.

## Structure

- `sprites/` - Ship sprites, islands, creatures, UI elements
- `sounds/` - Sound effects (cannons, water, impacts)
- `music/` - Background music tracks

## Asset Requirements

See README.md in the root directory for the complete list of required sprites and assets for this game.

## Notes

- All sprites should be PNG with transparency
- Recommended resolution: 2x for retina displays
- Sprite sheets should be horizontal strips
- Use consistent naming: `lowercase_with_underscores.png`
